# v0.0.6

A small follow-up to v0.0.5: a new full-screen generation UI, a nicer
thinking-reply glyph, and a sharper from-repo prompt. Three commits.

## Highlights

- **Full-screen animation for from-codebase generation** (`55ed9e1`) —
  replaces the single-line "Generating…" spinner with a dedicated Ink
  screen: a centered page that fills with drawn lines over time
  (scrolling once full, blinking cursor at the writing tip) layered over
  a matrix-style code rain seeded from the repo's own source. Elapsed
  timer in the bottom-right.
- **From-repo prompt tightened** (`64336f6`) — the prompt that generates
  a design doc from a codebase gained a size budget (~2k–4k words, ≤25
  sections) with a triage escape hatch for huge repos, breadth-first
  Features ordering, quotable-anchor guidance to avoid boilerplate
  section openings, permission to use short signature-shaped snippets
  (types, interfaces, flag lists, config schemas), jargon-on-first-use
  rules, and a terse final output reminder.

## Fixes

- **Thinking-reply glyph** (`350fe91`) — the interstitial reply posted
  while an agent is still working now renders as `🤖 is 🤔` instead of
  the previous placeholder, so commenters can tell at a glance that
  codocs has seen their comment and is thinking.
