# v0.0.3

Release-plumbing release. v0.0.2 uncovered gaps in the auto-publish path; this version fixes them so future version bumps actually ship binaries and npm packages without manual follow-up. No library or CLI behavior changes.

## Internal

- Shipped the compiled `dist/` artifacts alongside the npm packages — earlier releases were publishing metadata without the actual distributables (`0d389a9`).
- Added Windows binaries to the release artifact matrix (`43c4da1`).
- Bumped the Node version used by CI to keep the build reproducible (`12e7676`).
- Another pass at fixing the npm publish step after the v0.0.2 dry run (`0f3bb1d`).
- Bumped package versions to re-trigger the publish pipeline (`21813d3`).

**Full changelog:** https://github.com/gangeli/codocs/compare/v0.0.2...v0.0.3
